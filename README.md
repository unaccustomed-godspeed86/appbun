Turn any webpage into a desktop app using one simple command. appbun works on Windows, macOS, and Linux with tools designed to keep your apps small and fast.

## 🖥 What is appbun?

appbun helps you make desktop apps from websites. You do not need to code or learn new software. Just point appbun at a webpage, and it wraps it in a native app. It uses Electrobun, a lightweight option compared to other app makers. This means apps run smoothly without slowing your computer.

You can use appbun for:

- Running your favorite websites without opening a browser
- Keeping social media apps in separate windows
- Turning web tools into desktop apps for easy access
- Organizing webpages by task or topic on your desktop

## 🔍 Key Features

- Create desktop apps from any URL with one command
- Support for Windows, macOS, and Linux
- Small app size and fast start-up times
- Runs without needing a browser open
- Easy app management and removal
- Uses Electrobun, a modern alternative to Electron or Tauri
- No programming required

## 🎯 System Requirements

To use appbun on Windows, your PC should meet these minimum requirements:

- Windows 10 or later (64-bit)
- 4 GB of RAM or more
- 200 MB free disk space
- Internet connection during app creation
- Basic permission to install software

No special hardware or developer tools are needed.

## 🛠 Troubleshooting

If appbun does not start or shows errors:

- Make sure you have Windows 10 or later installed.
- Check that you have enough free disk space.
- Try restarting your computer and running appbun again.
- Disable any antivirus software temporarily, as it may block installation.
- Ensure your internet connection is stable when creating new apps.

## 📁 Where Are My Apps Stored?

By default, appbun saves your created desktop apps in:

```
C:\Users\<YourUser>\AppData\Local\appbun\apps
```

You can open this folder to find or move individual app files.

## 🛡 Security and Privacy

appbun places your desktop apps in isolated windows. It does not track your activity or store your browsing data beyond what is needed to run the apps.

Your created apps behave like regular desktop programs. Use common safety practices when downloading or running apps from the internet.
